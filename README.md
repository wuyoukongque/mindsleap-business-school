# MindsLeap Business School

Static course site for MindsLeap Business School.

## Local Preview

```bash
python3 -m http.server 8000 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:8000/
```

## Contents

- Course landing page and learning map
- Module pages
- Lesson pages
- Runtime CSS/JS
- Course configs, slides, subtitles, timelines, and production audio used by the site

Production source drafts, temporary audio chunks, local caches, virtual environments, and Vercel local bindings are intentionally excluded from Git.
